# Matching System

## Overview

The matching system is the core intelligence of TheWay. It compares user skills with opportunity requirements and generates a compatibility result.

---

## Purpose

The matching system helps users understand how close their profile is to a professional opportunity.

It highlights:

- Skills that match
- Skills that are missing
- Compatibility level
- Recommended improvements

---

## General Workflow

```txt
User Skills
    |
    v
Opportunity Requirements
    |
    v
Skills Comparison
    |
    v
Compatibility Score
    |
    v
Recommendations
```

---

## Matching Results

A matching result can include:

- Compatibility score
- Matched skills
- Missing skills
- Suggested skills to learn
- Opportunity relevance
