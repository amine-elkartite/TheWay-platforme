# Features

## Overview

TheWay provides a professional set of features focused on skills analysis, opportunity matching, user progression, and platform administration.

---

## User Features

- Account creation
- Login and logout
- Profile management
- Skills overview
- Opportunity browsing
- Matching score visualization
- Personalized recommendations
- Progress tracking

---

## Opportunity Features

- Opportunity listing
- Opportunity details
- Search and filtering
- Required skills display
- Compatibility score
- Company information

- User skills tracking
- Missing skills identification
- Recommended skills
- Market-oriented skills analysis
