# Dashboard

## Overview

The dashboard is the main workspace for users after authentication. It provides a clear overview of the user profile, skills, progress, and recommended opportunities.

---

## Purpose

The dashboard helps users quickly understand:

- Their current profile status
- Their main skills
- Their recommended opportunities
- Their matching progress
- Their improvement areas

---

## Main Sections

### Profile Summary

Displays basic user information and profile completion status.

### Skills Overview

Shows the user’s main skills and categories.

### Matching Summary

Displays compatibility indicators and recommended opportunities.

### Recommended Skills

Highlights skills that can improve the user’s career opportunities.

### Recent Activity

Displays recent actions, updates, or platform interactions.

---

## User Experience Goals

- Clear data presentation
- Modern card-based layout

- Professional visual hierarchy
- Simple and useful insights
